This project is to make the most money than buy-and-sold strategy

Using "Moving average" to accomplish the goal

If short-term average crosses long-term and downwards, it`s time to sell
If short-term average crosses long-term and upwards, it`s time to buy

MA: 5, 20, 60, 180 days

version 1 is only considered 5 days with others


require packages: numpy, pandas, matplotlib
